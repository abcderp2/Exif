Exif Remover Tool

JPEGおよびPNG画像からExifメタデータを削除するツールです。Web版とコマンドライン版の2つの使い方を提供しています。

🌐 Web版（オンラインツール）

今すぐブラウザで使用: https://abcderp2.github.io/Exif/

ドラッグ&ドロップで簡単操作

複数ファイルの一括処理

インストール不要

プライバシー保護（サーバーに画像を送信しません）

💻 コマンドライン版（開発者向け）

機能

JPEG、PNG画像のExifデータを安全に削除

単一ファイルまたは複数ファイルの一括処理

元のファイルを保護（新しいファイルとして保存）

エラーハンドリング機能

透明度を持つPNG画像にも対応

インストール

必要な環境

Python 3.9以上

依存ライブラリのインストール

pip install Pillow


ツールの取得

git clone https://github.com/abcderp2/Exif.git
cd Exif


使用方法

基本的な使い方

# 単一ファイルの処理
python exif_remover.py image.jpg

# 複数ファイルの処理
python exif_remover.py image1.jpg image2.png

# ワイルドカードを使った一括処理
python exif_remover.py *.jpg
python exif_remover.py photos/*.png


出力ファイル

処理された画像は元のファイル名に _no_exif が追加されたファイル名で保存されます。

例:

photo.jpg → photo_no_exif.jpg

image.png → image_no_exif.png

サポートするファイル形式

入力: JPEG (.jpg, .jpeg), PNG (.png)

出力: JPEG, PNG

実装されている機能

Web版・コマンドライン版共通

撮影日時情報の検出・除去: 撮影日時、作成日時、デジタル化日時

カメラ情報の検出・除去: メーカー、機種名、ソフトウェア情報

撮影設定情報の検出・除去: 露出、F値、ISO感度、焦点距離など

技術情報の検出・除去: 解像度、カラースペース、画像の向きなど

作者・著作権情報の検出・除去: 作者名、著作権表示

Web版のみの機能

詳細なExif情報表示: カテゴリ別に整理された見やすい表示

プライバシーリスク評価: 含まれる個人情報の種類を自動判定

リアルタイムプレビュー: アップロードした画像の即座表示

ファイル情報表示: サイズ、形式、画像サイズなどの基本情報

特徴

安全性: 元のファイルは変更されません

プライバシー重視: サーバーに画像を送信せず、ブラウザ内で完結処理

シンプル: 軽量で使いやすい設計

エラー処理: ファイルが存在しない場合や権限エラーなどを適切に処理

透明度保持: PNG画像の透明度は保持されます

品質保持: JPEG画像は高品質（95%）で保存されます

使用例

Web版

https://abcderp2.github.io/Exif/ にアクセス

画像をドラッグ&ドロップまたはファイル選択

Exif情報の詳細確認とプライバシーリスク評価

処理完了後にExif除去済み画像をダウンロード

コマンドライン版

# 写真フォルダ内のすべてのJPEG画像を処理
python exif_remover.py photos/*.jpg

# 特定のファイルを処理
python exif_remover.py vacation_photo.jpg profile_pic.png

# 処理結果の例
# Exifデータを削除しました: vacation_photo_no_exif.jpg
# Exifデータを削除しました: profile_pic_no_exif.png
# 処理完了: 2/2 ファイルが正常に処理されました

エラーハンドリング

このツールは以下のエラーを適切に処理します：

ファイルが見つからない場合

サポートされていないファイル形式

ファイル書き込み権限エラー

破損した画像ファイル

制限事項

位置情報（GPS情報）: 現在のバージョンでは位置情報の検出・表示機能は実装されていません

一部のExif拡張情報: 特殊なカメラメーカー固有のタグは表示されない場合があります
